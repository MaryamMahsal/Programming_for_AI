
# **🧪 Lab Activity 5: Mini Application – Student Performance Analysis using NumPy**

## **🎯 Objective**

To apply NumPy concepts for analyzing a dataset of student marks, including data creation, manipulation, statistical analysis, and normalization.

## **📌 Problem Statement**

You are given marks of students in 3 subjects (e.g., Math, AI, Programming). Use NumPy to analyze the dataset and generate useful insights.

## **📊 Step 1: Create Dataset**

Create a 2D NumPy array where: Rows represent students (15), Columns represent subjects (3)


## **📌 Step 2: Basic Array Properties**

* Display:

  * Shape
  * Number of dimensions
  * Data type

## **📌 Step 3: Indexing & Slicing**

* Extract:

  * Marks of 2nd student
  * Marks in 3rd subject
  * First 3 students
  * Last 4 students

## **📌 Step 4: Statistical Analysis**

Perform the following:

* Average marks per student
* Average marks per subject
* Highest and lowest marks per subject
* Top performing student

## **📌 Step 5: Mathematical Operations**

* Add 5 bonus marks to all students
* Increase marks of a specific subject

## **📌 Step 6: Conditional Operations**

* Find students scoring above 80
* Replace marks below 65 with 65

## **📌 Step 7: Extension Task (Challenge)**

Students should:

* Add more students dynamically
* Sort students based on average marks
* Visualize (optional, if matplotlib introduced)


## **📝 Submission Requirements in Google ClassRoom**

* Python script (`.py` or notebook)
* Output screenshots
* Brief explanation of results
