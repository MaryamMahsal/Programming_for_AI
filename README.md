# Lab Objective
The primary objective of this AI Lab is to provide students with hands-on experience in  programming practices relevant to Artificial Intelligence and to develop practical skills in  modern AI.-oriented programming environments. The lab focuses on implementing 
fundamental programming concepts, including the use of Integrated Development 
Environments (IDEs), variables, expressions, operators, control structures, loops, debugging, 
error handling, functions, strings, lists, and object-oriented programming constructs, with an 
emphasis on writing clean, efficient, and maintainable code. 
The lab also introduces contemporary AI-assisted development practices such as vibe coding, 
where students interact with AI coding assistants to generate, refine, and debug programs. In 
addition, students gain practical exposure to collaborative software development through the 
use of version control systems, learning to manage and track code using tools such as Git and 
Git-based repositories. Through guided exercises and small practical tasks, students develop 
the ability to implement, test, and manage AI-related programming workflows in a structured 
lab environment.
