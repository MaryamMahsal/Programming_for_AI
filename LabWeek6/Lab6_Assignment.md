# 📊 Lab Assignment: Student Performance Analyzer (Pandas)

## 🎯 Objective

The objective of this lab is to develop a mini-application using **Pandas** to analyze student performance data. Students will practice data handling, processing, and basic analytics.

---

## 📁 Dataset Description

You are provided with a CSV file named:

**`students.csv`**

It contains the following columns:

* Name → Student Name
* Math → Marks in Mathematics
* Physics → Marks in Physics
* CS → Marks in Computer Science

---

## 🛠️ Tasks to Perform

### 🔹 Task 1: Load the Dataset

* Import the Pandas library
* Load the dataset into a DataFrame
* Display the first few rows

---

### 🔹 Task 2: Data Processing

* Create a new column **Total** (sum of all subjects)
* Create a new column **Average**

---

### 🔹 Task 3: Grade Assignment

* Assign grades based on average marks:

  * A → ≥ 85
  * B → 70–84
  * C → 50–69
  * Fail → < 50

---

### 🔹 Task 4: Analysis

Perform the following:

* Identify the **top-performing student**
* Display all **failed students**
* Calculate **average marks for each subject**

---

### 🔹 Task 5: Data Filtering & Sorting

* Sort students based on Total marks (descending)
* Display top 3 students

---

### 🔹 Task 6: Save Results

* Save the updated dataset into a new CSV file:
  **`processed_students.csv`**

---

## 📌 Requirements

* Use **Pandas library**
* Write clean, readable code
* Use meaningful variable names
* Add comments to explain your logic
